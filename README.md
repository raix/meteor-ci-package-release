# meteor-ci-package-release
Npm module for releasing meteor packages from CI

# Create login token

```bash
$ meteor-ci create-token
# or with a key:
$ meteor-ci create-token --key superSecret
```

# Login via token

```bash
$ meteor-ci login j...iBhk=
# or with a key:
$ meteor-ci login j...iBhk= --key superSecret
```
